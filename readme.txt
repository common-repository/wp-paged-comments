WP Paged Comments

== Description ==

<p>这是一个针对Wordpress评论分页的增强插件。</p>

<h3>功能特性</h3>
<ul>
<li>评论分页， 可以选择所有的文章都分页还是特定的文章分页；</li>
<li>可以选择评论的显示排序， 是顺序还是倒序显示评论， 同样也可以指定特定文章相反方式排序；</li>
<li>无需对Wordpress和主题进行修改，便于安装。</li>
<li>可以选择每页最多评论条数；</li>
<li>可以自定义分页数导航条的样式；</li>
<li>可以选择最多显示的分页数， 超出部分用…表示， 可以改善排版；</li>
</ul>

<h3>Requirement to themes</h3>
<p>the comment objects of your theme must has an ID like “comment-xxx”. It's a standard of Wordpress themes now, yet some themes may still not meet this requirement.</p>

<h3>对主题的要求</h3>
<p>在主题中的评论的对象必须依从Wordpress主题标准，有“comment-xxx”这个ID，否则将无法使用。目前仍有少数主题不支持。</p>

== Installation ==
<ol>
<li>Download and unzip the Plugin package. </li>
<li>Upload it to Wordpress Plugin folder, '*WORDPRESSROOT*/wp-content/plugins/'</li>
<li>Activate from Plugin page in the Wordpress site admin. </li>
<li>Other configurations like CSS and more can be found in the option section in the site admin. </li>
</ol>

<h3>安装</h3>
<ol>
<li>下载并解压
<li>将此文件夹上传到Wordpress的插件目录：'Wordpress根目录/wp-content/plugins/'。</li>
<li>在后台的插件管理处激活。</li>
<li>其它设置，如CSS等请移步后台的设置区。</li>
</ol>
